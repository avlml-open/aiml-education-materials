# AIML Educational Materials

Public educational materials from AIML for community learning, brain-health literacy, and applied AI/ML education.

This repository is intended to hold handouts, slide decks, worksheets, facilitator notes, and related materials that AIML may share with community partners, educators, participants, and funders.

## Current materials

### Graceful Stretch / Brain-Healthy Activity Planner

A participant-facing handout for gentle movement and brain-health education. It includes:

- A 750 MET-point weekly activity planning framework
- Gentle Stretch Wellness class examples and add-on aerobic activities
- A weekly activity planner worksheet
- A MIND-style nutrition handout
- Cognitive engagement and Western North Carolina local-resource notes

See [`materials/graceful-stretch`](materials/graceful-stretch/).
